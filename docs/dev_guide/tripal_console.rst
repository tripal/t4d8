
Tripal Console
================

Tripal Console provides terminal-based commands for Tripal development and administration. Specifically, it uses `Drupal Console <https://drupalconsole.com/>`_ to provide an easy way to generate files when developing custom Tripal functionality through modules, fields and plugins.
